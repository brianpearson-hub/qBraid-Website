# Blog Images

This directory contains featured images for blog posts.

## Current Images Needed:

- `nsf-grant-header.jpg` - Header image for NSF Grant announcement post

## Image Guidelines:

- Recommended size: 1200x630px (Open Graph standard)
- Format: JPG or PNG
- Max file size: 500KB for optimal loading
- Use descriptive filenames
