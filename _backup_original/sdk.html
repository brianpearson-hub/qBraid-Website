qBraid SDK

Universal Python package for quantum computing

One SDK for All Quantum Providers

The qBraid SDK provides a unified interface for accessing quantum devices and simulators from multiple providers. Write code once and run it anywhere, with seamless transpilation between quantum frameworks and simplified job management.

pip install qbraid

Universal API

Single interface to access quantum devices from AWS, IBM, IonQ, and more without provider-specific code.

Circuit Transpilation

Automatically convert circuits between Qiskit, Cirq, Braket, PyTKET, and other frameworks.

Job Management

Unified job submission, tracking, and result retrieval across all quantum providers.

Device Discovery

Search and filter quantum devices by provider, topology, qubit count, and availability.

Open Source

Fully open source and community-driven. Contribute on GitHub or suggest new features.

Well Documented

Comprehensive API documentation with examples, tutorials, and guides for all features.
